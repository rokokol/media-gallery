# Attribution

Media Gallery is based on the original `obsidian-image-gallery` plugin by Luca Orio.

- Original project: `https://github.com/lucaorio/obsidian-image-gallery`
- Original plugin name: `Image Gallery`
- Original license: `MIT`

This version was adapted and extended by `rokokol` with support for mixed-media workflows, including explicit media lists, video and audio cards, audio playback, waveform and spectrogram previews, caching controls, and the `media-gallery` block alias.

Version lineage:

- `1.0.0`–`1.1.1` — original `obsidian-image-gallery` releases by Luca Orio
- `2.0.0` — first `Media Gallery` release by `rokokol`
